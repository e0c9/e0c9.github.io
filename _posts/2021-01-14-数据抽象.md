复合值将其他值结合起来：date: year, month, day

数据抽象使我们可以将复合值作为一个单元来操作

隔离使用数据程序的两个部分：

* 如何表示数据（as parts)
* 如何操作数据（as units)

数据抽象：在 representation 和 use 之间构建抽象屏障



有理数：分子，分母

* rational(n, d) 表示一个有理数x
* numer(x) 返回 x 的分子
* denom(x) 返回 x 的分母

以上实现了一个有理数的抽象表示。

```python
from fractions import gcd

def rational(n, d):
  g = gcd(n, d)
  return [n//g, d//g]

def numer(x):
  return x[0]

def denom(x):
  return x[1]

def add_rational(x, y):
  return rational(numer(x)*denom(y) + numer(y)*denom(x), denom(x)*denom(y))

```

抽象屏障:  使用有理数作为一个整体执行计算； 实现构造器和选择器constructor selectors

构造器与选择器一直指定正确的行为。

通过其行为来识别一个抽象数据表示。



用函数实现有理数：

```python
def rational(n, d):
  def select(name):
    '''此函数代表一个有理数'''
    if name == 'n':
      return n
    elif name == 'd':
      return d
  return select

def numer(x):
  return x('n')

def denom(x):
  return x['d']
```
