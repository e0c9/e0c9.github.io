打造高效的终端环境

Dec 25, 2020 2020-12-25T00:00:00+08:00 by 谭浩
Updated Jan 18 2021-01-18T23:02:46+08:00 2 min

合适的命令行工具（替换Mac自带的终端工具）：iterm2 + oh-my-zsh + zsh-autosuggestions

终端主题推荐：https://draculatheme.com/iterm/

zsh主题推荐：https://github.com/romkatv/powerlevel10k

推荐command tools:

  • https://github.com/ogham/exa is a modern replacement forls.
  • https://tldr.sh/ Simplified and community-driven man pages
  • https://github.com/tmux/tmux tmux is a terminal multiplexer
  • https://github.com/zsh-users/zsh-autosuggestions It suggests commands as you type based on history and completions.
  • https://www.imagemagick.org/script/convert.php Convert Between Image Formats
  • https://github.com/sindresorhus/speed-test Test your internet connection speed and ping using speedtest.net from the CLI
  • https://github.com/MrRio/vtop A graphical activity monitor for the command line.
  • https://github.com/junegunn/fzf#using-homebrew-or-linuxbrew fzf is a general-purpose command-line fuzzy finder
  • https://stedolan.github.io/jq/ jq is a lightweight and flexible command-line JSON processor.
  • https://web.hypothes.is/ START ANNOTATING or add it to your website.
