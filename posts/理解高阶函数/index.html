理解高阶函数

Jan 28 2021-01-28T00:00:00+08:00 by 谭浩
3 min

函数是一等公民

在函数式编程中函数为一等公民（first-class），一等公民具有以下特权：

  • 可以被绑定一个命名
  • 可以作为函数参数传递
  • 可以作为函数的结果进行返回
  • 可以被包含到数据结构中去

这些特性可以使我们在编程语言中将更为通用的计算过程表示为显式元素

函数作为参数

考虑数学上的求和运算，如何表示更通用的计算过程求1-10的平方的和，立方的和。

def summation(n, term):
  total, k = 0, 1
  while k <= n:
    total, k = total + term(k), k + 1
  return total
def cube(x):
  return x**3
def square(x):
  return x**2

def sum_cubes(n):
  return summation(n, cube)

函数作为返回值

Python中词法作用域的两个关键优势:

  • 局部函数的名称不会干扰定义该函数的外部函数的名称，因为局部函数名将绑定在其定义所在的当前局部环境中，而不是全局环境中。
  • 局部函数可以访问封闭函数(enclosing function)的环境，因为局部函数的主体是在扩展其被定义的环境的环境中进行求值的。

本地定义的函数维护着其parent environment

定义一个复合函数 h(x) = f(g(x))

2
3
4

def compose1(f, g): # enclosing function
  def h(x):
    return f(g(x))
  return h # 这是一个闭包，其包含了f,g的定义

Lambda表达式：定义一个函数而不需要给其命名。

1
2

def compose1(f, g):
  return lambda x : f(g(x)) # A function that takes and returns f(g(x))

Curring

可以利用高阶函数将一个带有多个参数的函数转化为一个链式函数每一个函数只需有一个参数。

1
2
3
4
5
6
7

def curried_pow(x):
  def h(y):
    return pow(y, x)
  return h # 一个闭包，包含了x的定义
  
cube = curried_pow(3)
square = curried_pow(2)

如何将一个带有两个参数的函数科里化

def curry2(f):
  def g(x):
    def h(y):
      return f(x, y)
    return h
  return g

函数装饰器

Python提供了特殊的语法，以在执行def语句（称为装饰器）的过程中应用高阶函数

def trace(fn):
  def wrapped(x):
    print('->', fn, '(', x, ')')
    return fn(x)
  return wrapped
  
@trace
def cube(x): # 等价于 cube = trace(cube), 把高阶函数应用到了执行def语句中
  return curried_pow(3)(x)
